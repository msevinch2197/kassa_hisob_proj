# Kassa Hisoblash Dasturi

Ushbu dastur mahsulot narxi va mijoz to‘lovini qabul qilib, quyidagi amallarni bajaradi:

## 📌 Funksiya haqida

`hisobla(float narx, float tolov)` funksiyasi:

- Agar to‘lov yetarli bo‘lsa, qaytimni hisoblaydi va chiqaradi
- Agar to‘lov yetarli bo‘lmasa, qancha pul yetishmayotganini ko‘rsatadi
- Agar noto‘g‘ri qiymat kiritilsa, xatolik haqida xabar beradi
